## Elements:
| Element | Type | Notes |
|---------|------|-------|
| شريط البحث | `AlhaiSearchField` | + زر Barcode |
| الفئات | `TabBar` scrollable | الكل = بدون فلتر |
| Product Card | Custom | صورة + اسم + سعر + badge مخزون منخفض |
| Cart Item | `ListTile` | اسم + سعر + quantity control + total |
| زر إتمام | `AlhaiButton.filled` | لون primary، كبير |

## Interactions:
- ضغط منتج → إضافة للسلة (أو زيادة الكمية إذا موجود)
- `+/-` → تعديل الكمية
- Swipe left → حذف من السلة
- إتمام البيع → يفتح Payment Sheet
